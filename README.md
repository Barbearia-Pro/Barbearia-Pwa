## 📱 Sobre o Projeto

O **Barbearia Pro PWA** é um sistema completo de gestão desenvolvido especialmente para barbearias, salões de beleza e profissionais autônomos que trabalham com agendamento.

Totalmente responsivo, funciona em qualquer dispositivo (celular, tablet ou computador) e não depende de internet para funcionar graças à tecnologia PWA (Progressive Web App).

### Por que escolher o Barbearia Pro PWA?

| Característica | Benefício |
|----------------|-----------|
| 🚀 **Funciona Offline** | Use mesmo sem internet no seu celular |
| 📱 **100% Mobile** | Interface adaptada para qualquer tela |
| 💬 **WhatsApp Integrado** | Confirme agendamentos com 1 clique |
| 💰 **Pagamento Único** | Sem mensalidades ou taxas escondidas |
| 🎨 **Design Premium** | Tema escuro profissional e moderno |
| 🔒 **Dados Seguros** | Tudo salvo localmente no dispositivo |
| 📡 **Tecnologia PWA** | Instalável como aplicativo nativo |
| ⚡ **Rápido e Leve** | Carrega instantaneamente |

## 🎯 Funcionalidades

### 📅 Módulo de Agendamentos
- Calendário visual por data
- Verificação automática de conflitos de horário
- Seleção rápida de cliente e serviço
- Confirmação de agendamento via WhatsApp
- Cancelamento de agendamentos
- Histórico completo

### 👥 Módulo de Clientes
- Cadastro completo (nome, telefone, e-mail, data de nascimento)
- Histórico de todos os atendimentos
- Campo de observações e preferências
- Busca inteligente por nome ou telefone
- Integração direta com WhatsApp
- Exclusão de cadastro

### 💇 Módulo de Serviços
- Catálogo de serviços ilimitado
- Cadastro de preço e duração
- Descrição detalhada de cada serviço
- Serviços padrão pré-cadastrados
- Exclusão de serviços

### 💰 Módulo Financeiro
- Registro de pagamentos
- Múltiplas formas: Dinheiro, Cartão (Crédito/Débito), PIX
- Faturamento do dia em tempo real
- Faturamento mensal automático
- Filtro por data
- Totalizador diário

### 📊 Dashboard Principal
- Resumo dos agendamentos do dia
- Faturamento diário em destaque
- Total de clientes cadastrados
- Atualização automática a cada minuto
